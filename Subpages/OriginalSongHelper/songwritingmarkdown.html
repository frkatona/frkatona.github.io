Formatting Help

To format your song, use the following rules:

  • Write your lyrics as normal text.
  • To include chords, place the chord names in square brackets within the lyrics, like this: [C], [G], [Am], etc.
  • To define sections, start a line with the section name followed by a colon, like Verse:, Chorus:, Bridge:, etc.
  • The chords will be displayed above the corresponding lyrics.
